# Sales Team Tools

Monorepo containing the Sales Chrome Extension and its Admin Dashboard.

## Projects

### `sales-extension/`
Chrome Extension (MV3) for generating personalized cold emails and LinkedIn messages using AI. Built with React, TypeScript, and Vite.

### `sales-admin/`
Admin Dashboard for managing extension settings, prompts, users, and API configuration. Built with Nuxt 3, Tailwind CSS, and Cloudflare Pages + D1.
